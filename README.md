# 🎸 Band-man Task  
バンドマン向けタスク管理iOSアプリ

---

## 📱 概要
**Band-man Task** は、軽音部やサークルなどでバンド活動を行う人向けに開発した  
**楽曲管理・ライブ管理を一体化したタスク管理アプリ**です。

バンド活動では、
- 演奏する曲の管理
- 練習の進捗把握
- ライブ予定やセットリストの整理

といった情報を複数の場所で管理しがちで、把握が煩雑になるという課題があります。  
本アプリではそれらを一元管理し、**可視化することで練習や準備を効率化**することを目的としています。

---

## 🎼 主な機能

### ① 楽曲管理
- 曲の登録 / 編集 / 削除
- 完成度（練習進捗）の記録
- 各パート（Gt / Ba / Dr など）の登録
- YouTubeリンクの登録

### ② ライブ管理
- ライブ日程の登録 / 編集 / 削除
- セットリストの管理

### ③ ダッシュボード機能
- 直近のライブ予定を表示
- 完成度の低い曲を自動でピックアップ
- 楽曲・ライブの情報をもとにホーム画面を動的に構成

## 💡 工夫した点

### ユーザーにとって分かりやすいUI
- タブバー構成（ホーム / 曲 / ライブ）
- ソート・検索機能の実装
